# CSharp-MathParser-CLI

C#の基礎やアーキテクチャ設計を深く学ぶために、ゼロから自作した「構文解析エンジン」および「CLI電卓」のプロジェクトです。

## プロジェクトの目的
計算ロジック、C#の言語仕様やオブジェクト指向への理解を深めることを目的とした学習用リポジトリです。

## 特徴
* **自作の構文解析エンジン:** 既存の計算用ライブラリにできるだけ頼らず、入力された文字列から数式を解析して計算するロジックを独自実装しています。
* **UIとロジックの完全分離:**
  * `CoreEngine/` (クラスライブラリ): 純粋な計算・解析ロジックのみを担当。
  * `CalcCLI/` (コンソールアプリ): ユーザー入力の受け取りと画面表示のみを担当。

## 構成
* `CsLearning.sln` : ソリューションファイル
* `CoreEngine.csproj` : 構文解析エンジン
* `CalcCLI.csproj` : 電卓アプリ

## 使い方
1. `CsLearning.sln` を Visual Studio 等で開く。
2. スタートアッププロジェクトを `CalcCLI` に設定する。
3. ビルドして実行し、コンソール画面に数式を入力して動作を確認してください。
